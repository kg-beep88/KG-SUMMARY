KG SIMPLE SITE CALENDAR V4
==========================

FOR EXISTING LIVE WEBSITE
-------------------------
1. Use the SAFE-UPGRADE ZIP.
2. Extract it.
3. Upload all files to the root of the GitHub repository.
4. Replace the old files.
5. Do not delete your current supabase-config.js.
6. Commit changes.
7. Open the website and press Ctrl + F5.

HOW TO USE
----------
Calendar > choose date > Site entry

Choose:
- Address
- Worker / Foreman / Subcon / All
- Active / Pause / Claim / Complete
- Work or Work cannot be done
- Equipment needed

Use the buttons for:
- Delivery to site
- Stock return from site
- Ladder / equipment borrow and partial return

No Google Calendar setup is needed.
